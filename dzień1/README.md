# Dzień 1

## Treść zadania
Napisz metodę, która zwróci ilość monet, którą należy zwrócić dla zadanej wartości.

Monety w nominałach 100, 25, 10, 5, 1

## Uruchomienie
> dotnet run -- wartość

## Przykład 
> dotnet run -- 457

```
Reszta z 457
Suma wszystkich monet 9
100: 4
 25: 2
 10: 0
  5: 1
  1: 2
```
